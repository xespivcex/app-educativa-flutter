MANTY: Aprendo y Manejo mi Dinero
MANTY es una aplicación móvil inclusiva desarrollada en Flutter, creada con el objetivo de fortalecer la alfabetización y la educación financiera básica. Este proyecto nace como una herramienta pedagógica para apoyar a niños y adultos en el desarrollo de su autonomía económica.

Contexto del Proyecto
La aplicación está siendo desarrollada para un programa piloto, enfocándose en la creación de una interfaz sencilla, accesible y adaptada a las necesidades locales de aprendizaje financiero.

Funcionalidades Actuales
La versión actual del prototipo incluye los siguientes avances:

Selección de Perfiles: Diferenciación de la experiencia para tres tipos de usuario: Niño, Adulto y Tutor.

Módulos Educativos:

Lectura: Evaluación de asociación de palabras y letras.

Números: Identificación y conteo básico.

Dinero: Introducción a la suma de valores monetarios.

Compras: Simulación de una transacción real donde el usuario debe decidir si el dinero le alcanza para un producto.

Panel de Progreso: Visualización de logros y puntajes mediante indicadores gráficos (estrellas y barras de porcentaje).

Aspectos Técnicos
Lenguaje: Dart.

Framework: Flutter.

Interfaz (UI): Diseño basado en componentes de alto contraste y botones de gran tamaño para asegurar la accesibilidad.

Lógica de Juego: Sistema de validación de respuestas en tiempo real con retroalimentación visual (✅/❌).

Criterios de Accesibilidad
Para asegurar que la app sea utilizada por la mayor cantidad de personas, se han aplicado los siguientes criterios:

Navegación Simple: Menús lineales y directos.

Apoyo Visual: Uso de iconos representativos para cada acción.

Tamaño de Fuente: Textos grandes y legibles.
